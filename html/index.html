    Cooperation

    NHN NEXT라는 곳은 여러 다른 전공들이 모여 개발이라는 분야를 배우는 교육기관입니다. 경영학도도 있었고, 디자인하는 사람들도 있었습니다. 함께 수업을 듣고 학습하며 자연스럽게 이들의 사고 방식을 이해하고, 제 생각의 틀을 넓힐 수 있는 기회가 되었습니다. 이러한 점들이 앞으로 프로젝트를 진행하는데 있어서, 여러 포지션들과의 대화에 큰 도움이 될 것이라고 확신합니다.

    Coding

    프로그래밍 언어의 모태라고 할 수 있는 C언어와 객체지향언어의 대표주자인 JAVA 언어를 배웠습니다. Java로는 Servlet과 JSP를 이용하여 게시판을 만드는 프로젝트를 진행하였고, Spring에 대한 기초적인 이해를 바탕으로 프로젝트를 진행하였습니다. Front-End 분야에 있어서는 HTML5, CSS3 그리고 JavaScript, jQuery 등에 대한 여러 프로젝트를 진행하였고, 현재 React에 대한 학습을 하고 있습니다.

    UX Design

    디자인 프로젝트라는 과목을 통하여, UX에 관련된 기본적인 지식을 습득하였습니다. 그에 그치지 않고 실제 서비스를 기획하는 이 프로젝트를 통해 팀원과 함께 브레인스토밍을 통한 서비스 기획과정 부터, 직접 설문조사를 실시하고, 자료조사를 하고, 최종적으로는 툴을 사용하여 프로토타입을 제작하는 등 UX에 대한 실전적인 내용을 학습하였습니다.

    Todo MVC project I

    To do MVC - VanilaJS version

    JavaScript를 공부하면서, MVC 구조를 연습하기 위해 TODO MVC를 만들어 보았습니다. 웹 애플리케이션을 구축할 때 쓰이는 아키텍쳐인 MVC 모델을, 일반 애플리케이션을 만드는데 적용해보면서, Model, View 그리고 Controller의 역할을 명확히 하는데 집중했습니다. 그 과정을 블로그에 단계를 나누어 포스팅하였습니다. 공식 홈페이지도 있는 유명한 Todo MVC가 홈페이지에는 초심자 입장에서는 너무 어려운 코드들로 작성되어 있었고, 아무런 설명없이 코드만 달랑 있기에, 처음부터 차근 차근 설명하면서 자바스크립트를 공부하는 사람이라면 누구나 시도해볼 수 있도록 포스팅했습니다.
